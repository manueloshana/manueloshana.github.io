# Manuel Oshana Portfolio

A static portfolio website for Manuel Oshana, styled as a VS Code workspace inside a Windows-style application window.

## Files

- `index.html` is the main portfolio page.
- `script.js` contains the editable project data.
- `styles.css` contains the visual system and responsive layout.
- `resume.html` is a printable resume page.
- `assets/manuel-oshana-atelier.png` is an unused generated asset from the first design pass.

## Edit Your Projects

Open `script.js` and update the `files` object. The portfolio is organized as editable virtual files:

- `about.md`
- `projects.ts`
- `resume.json`
- `contact.http`

## Open Locally

Open `index.html` in a browser. No build step is required.
